# claude-container のデフォルト pip パッケージ一覧。
# プロジェクト側の .claude-container.d/requirements.txt が無い場合にのみ
# フォールバックとして使用される。コンテナのシステム Python
# （pip3 install --break-system-packages）にそのまま渡される。
#
# 多くのプロジェクトでは Python 依存はプロジェクト側の venv 等で隔離管理される
# ため、通常は空のままでよい。venv の外（システム Python）で動くスクリプトが
# 必要とする追加パッケージがある場合のみ、プロジェクト側の
# .claude-container.d/requirements.txt に 1 行 1 パッケージで記載する。
