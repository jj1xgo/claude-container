# claude-container のデフォルト追加許可ドメイン一覧。
# プロジェクト側の .claude-container.d/allowed-domains.txt が無い場合にのみ
# フォールバックとして使用される。1行1ドメイン、# 始まりはコメント。
#
# ここに書いたドメインは init-firewall.sh がエグレス許可リストへ追加する
# （Claude Code に必要な api.anthropic.com・GitHub 等は init-firewall.sh 側で
# 常に許可されるため、ここに書く必要はない）。
#
# claude-container は汎用ツールのため、ここにプロジェクト固有のドメインを
# 書かないこと。プロジェクト固有のドメイン（例: pypi.org）は各プロジェクト側の
# .claude-container.d/allowed-domains.txt に記載する。
# 変更をイメージへ反映するには -b での再ビルドが必要。
